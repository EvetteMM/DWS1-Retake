Element Definitions

Evette Mestre Meyer

<h1>
The main heading of a page and generally the largest font on the page
<h2>
A level 2 heading. Usually denotes a main topic
<h3>
A level 3 heading, a sub-heading
<h4>
A level 4 heading, a subheading
<h5>
A level 5 heading, a subheading
<h6>
A level 6 heading, the lowest subheading in HTML
<p>
Denotes a paragraph
<b>
Denotes boldface type
<i>
Denotes italic type
<sup>
Denotes superscript used in dates and exponents, ie: February 4th or 1010
<sub>
Denotes subscript used in footnotes or chemical formulas, ie: CO2
Whitespace
A browser only recognizes the first space in a group of consecutive spaces so designers use this to better organize their code.
<br>
A line break to move text down to the next line or to add a blank line.
<hr>
Adds a dividing line to a web page for visual, organizational purposes
<strong
Screen readers and search engines recognize strong text as something of greater importance or relevance on a page. Often appears as boldface type on a web page.
<em>
Screen readers and search engines recognize strong text as something of to be emphasized or of importance. Often appears as italic type on a web page.
<blockquote>
Screen readers and search engines often automatically indent a blockquote on a page. It is still important to add indentation to the corresponding CSS page.
<q
Screen readers and search engines recognize this for shorter quotes that can be part of a line of text.
<abbr>
Denotes an abbreviation of a title. Has also replaced the acronym element.
<cite>
Denotes a citation of a literary work, movie, etc. Appears as italic.
<defn>
Screen readers and search engines recognize this as a definition or new term on a page
<address>
Holds the contact information for the author of the page, ie: email address and/or physical address
<ins>
The insert element inserts a word/phrase into a line and denotes by underlining it
<del>
The delete element deletes a word/phrase from a line and denotes it by putting a line through it
<s>
Adds a strike-through line to text, ie: to show a price has been slashed
<ol>
Signifies an ordered list where the list items are noted in numerical or alphabetical order
<ul>
Signifies an unordered list where the list items are noted with bullet points, squares, diamonds or an image
<li>
A list item in either an ordered or unordered list
<dl>
A definition list
<dt>
A definition term
<dd>
A definition description
<Nested List>
A list item attached to another list item
